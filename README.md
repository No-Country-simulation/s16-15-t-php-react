##  Web app de FRELANCER-- WORKFRELANCER 



##  ¿Cuál fue el propósito de crear WORKFRELANCER ?  
Creemos que la creacion de WORKFREELANCER, los freelancers necesitan una herramienta de búsqueda avanzada y filtros específicos para encontrarue que se ajusten a sus habilidades y disponibilidad. Esto se hizo con la intención de que los freelancers encuentren más proyectos relevantes y apliquen a más trabajos utilizando la herramienta de búsqueda avanzada.

## ¿Qué busca la web app de WORKFRELANCER ?
La web app de WORKFREELANCER busca abordar varios desafíos que enfrentan tanto los freelancers como las empresas y empleadores. Para los freelancers, la web app busca resolver la dificultad para encontrar proyectos relevantes y la falta de visibilidad y reconocimiento. Esto se logra mediante la creación de una plataforma con un motor de búsqueda avanzado y filtros específicos, así como la oferta de perfiles detallados y personalizables con la posibilidad de incluir portafolios.

Para las empresas y empleadores, la web app busca abordar la dificultad para encontrar talento cualificado y la dificultad para visualizar candidatos. Esto se logra implementando una herramienta de búsqueda y una tarjeta con la información del freelancer resumida. Además, se ofrece la posibilidad de visualizar a un candidato específico en una sección dedicada para consultar los datos de la tarjeta y la propuesta final
